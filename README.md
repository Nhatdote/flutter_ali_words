| [english_words](https://breakdance.github.io/breakdance/) | Fluuter package: Make random english word |

## Installing
_Make sure you have installed Flutter already_
```sh
git clone git@github.com:Nhatdote/ali_words.git
cd ali_words
flutter pub get
flutter run
```
